# Shortcuts
## Window
- hyper + f: fullscreen
- hyper + =: increase window size
- hyper + -: increase window size
- hyper + -> : move window right
- hyper + <- : move window left

## App
- hyper + c: focus Google Chrome
- hyper + t: focus Terminal
- hyper + a: focus Finder

## Clipboard
- hyper + v: clipboard modal
